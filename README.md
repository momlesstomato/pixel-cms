# Pixel CMS

Pixel CMS is a typed Next.js application scaffolded around the project standards in `AGENTS.md`.

## Scripts

```bash
npm run dev
npm run lint
npm run test
npm run typecheck
npm run build
```

## Structure

Routes stay thin and render views from `src/views`.

```text
src/app/page.tsx
src/views/Home/Home.tsx
src/views/Home/Home.test.tsx
```
